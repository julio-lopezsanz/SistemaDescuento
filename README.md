# Sistema de Descuentos en Python

Este proyecto es un sistema de descuentos desarrollado en Python.  
Aquí aplico los conceptos que he aprendido hasta ahora sobre programación y lógica.

El objetivo es simular cómo funcionan los descuentos en una tienda real utilizando reglas programables.

---

## ¿Qué hace el programa?

El programa solicita información sobre una compra y calcula el precio final aplicando descuentos según las siguientes reglas:

- 10% de descuento si el cliente es VIP.
- 5% de descuento adicional si el cliente compra 12 o más productos.
- Descuento basado en el subtotal:
  - Más de $1000 → 20%
  - Más de $2000 → 30%
  - Más de $3000 → 40%

Los descuentos son acumulables.

---

## ¿Qué datos solicita el programa?

- Nombre del producto
- Precio del producto
- Cantidad comprada
- Si el cliente es VIP

---

## ¿Qué conceptos de Python practiqué?

- Manejo de variables y tipos de datos
- Entrada de datos por teclado (`input`)
- Validación de datos
- Manejo de errores (`try/except`)
- Estructuras de control (`if`, `elif`, `else`)
- Operadores lógicos, aritméticos y comparativos
- Cálculo de porcentajes
- Formateo de salida
